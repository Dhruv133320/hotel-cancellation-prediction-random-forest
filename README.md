# Hotel Booking Cancellation Prediction using Random Forest

## Overview

Hotel booking cancellations can significantly impact revenue, resource planning, and customer experience. This project applies machine learning techniques to predict whether a hotel booking is likely to be cancelled based on customer and reservation details.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training using the Random Forest algorithm, and performance evaluation.

---

## Business Problem

Hotels experience substantial financial losses due to unexpected booking cancellations. Accurately predicting cancellations allows hotels to:

- Improve occupancy planning
- Reduce revenue loss
- Optimize pricing strategies
- Design targeted customer retention campaigns

This project aims to identify cancellation patterns and build a predictive model that supports data-driven decision-making.

---

## Objectives

- Clean and preprocess hotel booking data
- Perform exploratory data analysis (EDA)
- Engineer relevant features
- Train a Random Forest classifier
- Evaluate model performance
- Identify the most influential features affecting cancellations
